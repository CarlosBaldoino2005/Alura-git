## Requisitos

| Item | Detalhe |
|------|---------|
| IDE | Embarcadero Delphi (versão compatível com o projeto) |
| SO | Windows (recomendado para projetos VCL/FMX) |
| Dependências | Ver arquivos do projeto (`.dproj`, `boss.json`, etc.) |

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/CarlosBaldoino2005/Alura-git.git
   ```
2. Abra o projeto (`.dproj` ou `.dpr`) no Delphi.
3. Compile e execute conforme as instruções do código-fonte.
